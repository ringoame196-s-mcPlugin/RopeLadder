# RopeLadder

## プラグイン説明

縄梯子を追加するプラグイン

## 使い方

はしごをインベントリに持った状態のままシフトしながら はしごを設置

## configファイル

- max-size - 縄梯子の最大ブロック数

## 開発環境

- Minecraft Version : 1.20.1
- Kotlin Version : 1.8.0
